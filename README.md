# Python Basics - Complete Beginner's Guide

A collection of Python code examples from the Medium article: **"Python Basics for Beginners: Your Complete Starting Guide"**

## 📚 What's Included

This repository contains all the code examples covered in the article:

- ✅ Your first Python program using `print()`
- ✅ Variables and basic data types
- ✅ Getting user input with `input()`
- ✅ Formatting output with f-strings
- ✅ Arithmetic operators and type conversion
- ✅ Mini projects to practice

## 📁 Files

| File | Description |
|------|-------------|
| `hello.py` | Your first Python program |
| `cal.py` | Calculator mini project - arithmetic operations |
| `function.py` | This program uses multiple functions to generate student report card |

## 🚀 How to Use

1. **Clone this repository:**
   ```bash
   git clone https://github.com/prakashgangurde-ux/python-basics.git
   cd python-basics
   ```

2. **Run any example:**
   ```bash
   python hello.py
   python cal.py
   ```

3. **Experiment:** Feel free to modify the code and run it again!
